# PromiseA+

用 javascript 和 typescript 手写 promiseA+规范

## 规范

规范文档 [promises-spec](https://github.com/promises-aplus/promises-spec)

测试插件 [promises-tests](https://github.com/promises-aplus/promises-tests)

## 运行

- 安装依赖

```bash
yarn install
```

- 仅测试 js

```shell
  npm run test:js
```

- 仅测试 ts

```shell
  npm run test:ts
```

- 测试全部

```shell
  npm run test
```
